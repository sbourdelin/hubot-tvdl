# hubot-tvdl

A hubot script that control the TVDL

See [`src/tvdl.coffee`](src/tvdl.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-tvdl --save`

Then add **hubot-tvdl** to your `external-scripts.json`:

```json
[
  "hubot-tvdl"
]
```
